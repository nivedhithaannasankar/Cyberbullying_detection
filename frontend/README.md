# Cyberbullying Detection System - Frontend

## Overview

The Cyberbullying Detection System is a web-based application developed using React.js that helps identify and classify cyberbullying content in text messages. Users can enter text, submit it for analysis, and view the prediction results through an intuitive user interface.

This frontend application provides a responsive and user-friendly interface for interacting with the cyberbullying detection model.

---

## Features

* User Authentication Interface
* Text Input for Cyberbullying Detection
* Real-Time Prediction Results Display
* Responsive User Interface
* API Integration with Backend Services
* Clean and Interactive Dashboard
* Error Handling and Validation

---

## Installation

### Clone the Repository

```bash
git clone <repository-url>
```

### Navigate to Project Directory

```bash
cd cyberbullying-detection-frontend
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```

---

## Build for Production

```bash
npm run build
```

The optimized production build will be generated inside the `build` folder.

---

## API Configuration

Update the backend API URL inside:

```text
src/api.js
```

Example:

```javascript
const BASE_URL = "http://localhost:5000";
```
